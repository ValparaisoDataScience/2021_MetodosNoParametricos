Conceptos importantes

La estadística multivariante o multivariada es una rama de las estadísticas que abarca la observación y el análisis simultáneos de más de una variable respuesta. La aplicación de la estadística multivariante es llamada análisis multivariante.

Los análisis multivariantes son muy utilizados en Ciencias del Mar y Oceanografía como herramienta para realizar análisis exploratorio de datos y detectar variables latentes.

Análisis de agrupamiento o “cluster”
Los análisis de agrupamiento o clustering en ingles tiene como objetivo agrupar datos (muestras) por similitud, en grupos o conjuntos de manera que los miembros del mismo grupo tengan características similares Wiki.

Análisis de componentes principales
Los análisis de componentes principales (en español ACP, en inglés, PCA) son una técnica utilizada para describir un conjunto de datos en términos de nuevas variables («componentes») no correlacionadas. Los componentes se ordenan por la cantidad de varianza original que describen, por lo que la técnica es útil para reducir la dimensionalidad de un conjunto de datos Wiki..

Matriz de distancia o similaridad
Subyacente a muchos de los análisis multivariantes está el uso de matrices de distancia o similaridad por lo que su calculo y adecuada interpretación es fundamental previo a la realización de los análisis multivariantes.

Una matriz de distancias es una matriz cuyos elementos representan las distancias entre los puntos, tomados por pares, de un conjunto. Se trata, por lo tanto, de una matriz simétrica de tamaño NxN.

Matriz de distancia euclideana
La distancia euclidiana, es la distancia entre dos puntos de un espacio euclídeo, la cual se deduce a partir del teorema de Pitágoras Wiki.

Comandos para realizar los análisis.

read_excel()
Importa datos a R desde archivos excel.

select Selecciona variables desde un data.frame o una tibble.

%>% El operador pipe sirve para concatenar tareas o funciones.

mutate() Crea nuevas variables a partir de variables originales de un data.frame o una tibble.

t() Dada una matriz x, t() entrega la transpuesta x.

dist() Calcula diferentes matrices de distancia entre las filas de un conjunto de datos.

vegdist Calcula diferentes medidas de distancia o similaridad (usualmente para datos de ecología) entre las filas de un conjunto de datos.

Ejercicio 2. Matriz distancia euclideana

  1. Para las tres variables cuantitativas continuas Depth, Pollution y Temperature elabore 3 variables derivadas D1, P1 y T1 con su valor estandarizado como la diferencia de cada valor por la media mean() y dividiendo por la desviasión estandar sd(). Use la función mutate() de la librería dplyr
# Crea nuevas variables
val_estandarizado <- bioenv %>%
 select(Sitio, Depth, Pollution, Temperature) %>%
 mutate(
  D1 = (Depth - mean(Depth)) / sd(Depth),
  P1 = (Pollution - mean(Pollution)) / sd(Pollution),
  T1 = (Temperature - mean(Temperature)) / sd(Temperature))
  1. Elabore una gráfica de correlaciones de las variables estandarizadas usando la función pairs.panels() y compare con las correlaciones estimadas en el ejercicio 1b que calcula la correlación con la variable original.
